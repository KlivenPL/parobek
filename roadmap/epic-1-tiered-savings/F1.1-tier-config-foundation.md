# F1.1 — Tier config foundation

- [x] **Status:** done
- **Tier:** 0 (infra) · **Phase:** NOW · **Epic:** 1

## Goal

Add the `savingTier` setting and the tier presets/helpers that every other feature
reads to decide whether it is active and whether to warn.

## Mechanism

Clone the endpoint-`PRESETS` pattern already in
[`config.mjs`](../../plugins/parobek/scripts/lib/config.mjs):

- `TIER_PRESETS` (code-owned), parallel to `PRESETS`:
  ```js
  export const TIER_PRESETS = {
    safe:     { level: 1, name: 'Safe',     features: ['mcp-digest', 'mcp-codesearch', 'local-commit', 'local-index'] },
    balanced: { level: 2, name: 'Balanced', features: [/* …safe + */ 'file-predigest', 'big-read-nudge', 'tier-guidance', 'local-handoff'] },
    max:      { level: 3, name: 'Max',      features: [/* …balanced + */ 'big-read-substitute', 'auto-precompact', 'local-ask'] },
  }
  ```
- `DEFAULT_CONFIG.savingTier = 'safe'`.
- In `readConfig`, merge custom tiers from the file (built-ins win the base, user
  extends): `tiers: { ...TIER_PRESETS, ...(parsed.tiers ?? {}) }`.
- In `writeConfig`, strip built-in tier names exactly like built-in presets are
  stripped, so the file never carries a dead copy and legacy blocks self-clean.
- New helpers:
  - `resolveTier(config) → { key, level, name }` — reads `config.savingTier`, falls
    back to `safe` for unknown values.
  - `featureEnabled(config, featureId) → boolean` — `level >= min tier of feature`.
    Backed by a `FEATURE_TIER` map (featureId → min level).
  - `commandTierExceeds(config, cmdLevel) → boolean` — `cmdLevel > resolveTier().level`.
- Extend `configSummaryLines` with a `savingTier:` line (e.g. `balanced (Balanced, lvl 2)`).

## Files

- **Change:** [`scripts/lib/config.mjs`](../../plugins/parobek/scripts/lib/config.mjs)
  — `TIER_PRESETS`, `FEATURE_TIER`, `DEFAULT_CONFIG.savingTier`, merge/strip in
  read/write, helpers, `configSummaryLines`.

## Feasibility / risk

None — pure config. The merge/strip logic is already proven for endpoint presets.

## Acceptance & verification

- Unit tests (`scripts/__tests__/`): `resolveTier` for each preset + unknown;
  `featureEnabled` across levels; `commandTierExceeds`; round-trip a custom tier
  through `writeConfig`/`readConfig` and assert built-ins are not persisted.
- `node scripts/local-config.mjs status` shows the `savingTier` line.
