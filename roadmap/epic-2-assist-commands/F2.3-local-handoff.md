# F2.3 — `/local-handoff`

- [ ] **Status:** not started
- **Tier:** 2 (Balanced) · **Phase:** NOW · **Epic:** 2

## Goal

Hand work to a fresh agent/session: search the current conversation for what's
relevant to a given prompt, summarize it locally into a focused handoff brief, and
inject that brief into the next window. Like `/local-compact`, but **prompt-scoped**
rather than a full summary.

**Tier 2 rationale:** the new session reasons off a *local* brief (filtered input),
so a relevant detail could be dropped.

## Mechanism

```
/local-handoff <prompt>      # what the next agent should focus on
```

- `scripts/local-handoff.mjs`:
  1. `warnIfTierExceeds` (tier 2).
  2. Preflight + read the recorded transcript (`readSessionRecord` →
     `parseTranscript`).
  3. Ask the local model to extract **only what's relevant to `<prompt>`** as a
     handoff brief (`lib/handoff-prompt.mjs`): task framing, decisions made, current
     state, open threads, key files — scoped to the prompt. Reuse the
     single-pass / map-reduce structure from `local-compact.mjs`.
  4. Persist through the **existing pending-summary machinery**
     (`writePendingSummary`) with `kind: 'handoff'` + the focus prompt stored
     alongside.
  5. Tell the user to run `/clear` to start the scoped session.
- On the next `/clear`, [`inject-summary.mjs`](../../plugins/parobek/scripts/inject-summary.mjs)
  injects it under all five safety guards. Minor change there: when
  `pending.kind === 'handoff'`, adjust the injected header to read "Handoff brief
  (focus: …)" instead of the compact-summary header.

## Files

## Feasibility / risk

- Reuses a proven flow; the only new surface is the focus-scoped prompt and the
  `kind` field. All five injection guards carry over unchanged.
- Same correlation-by-cwd constraint as `/local-compact` (documented in
  `state.mjs`).

## Acceptance & verification

- Integration: record a session, run `/local-handoff "<focus>"`, assert a pending
  summary with `kind: 'handoff'` exists; then feed a `SessionStart` `source: clear`
  event and assert the handoff header is injected and the pending file is consumed.
